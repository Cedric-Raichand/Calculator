# Simple Calculator Web App

This is a **basic calculator web application** built using **HTML, CSS, and JavaScript**.  
It performs standard arithmetic operations and features a clean, user-friendly interface.

The project was created for **practice with DOM manipulation, event handling, and CSS grid layouts**.

---

## ✨ Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear (C) button
- Decimal number support
- Responsive button layout using CSS Grid
- Error handling for invalid expressions
